# Funktionale Programmierung vs. imperative Programmierung

## Haskell

### Eingabe

```file
path: code/hello.hs
lang: haskell
```

### Ausgabe

```terminal2
runhaskell code/hello.hs
```

---

## Python

### Eingabe

```file
path: code/hello.py
lang: python
```

### Ausgabe

```terminal2
python code/hello.py
```

---

## Java

### Eingabe

```file
path: code/hello.java
lang: java
```

### Ausgabe

```terminal2
java code/hello.java
```
